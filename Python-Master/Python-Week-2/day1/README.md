# Day 1: Arrays and Two Pointers Basics

## Focus
We are entering into data structures. Arrays (Lists in Python) are the fundamental building block. Today focuses on understanding how arrays work in memory, basic traversals, and the essential **Two-Pointer Technique**.

## Learning Objectives
1. Understand Python List internals (dynamic arrays, memory allocation).
2. Master array traversal (forward, backward, jumping).
3. Recognize scenarios for the Two-Pointer pattern (opposite ends, same direction).
4. Analyze Time & Space complexity for array operations ($O(1)$ vs $O(N)$ vs $O(N^2)$).

## Drills
- Reversing arrays in-place.
- Two Sum (when the array is sorted).
- Removing duplicates / elements in-place.
- Moving Zeroes while maintaining order.
- Merging sorted arrays.
- Squaring a sorted array and keeping it sorted.
- Finding intersection and union.
- Finding majority element and missing element.
- Pair and triplet sums.
