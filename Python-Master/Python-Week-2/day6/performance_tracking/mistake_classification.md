# Mistake Classification Log

Use this document to categorize the bugs you encountered during the Marathon and Mock test.

## Categorizations:
1. **Logic Error**: Fundamental misunderstanding of the algorithm/pattern.
2. **Off-by-one**: `index out of bounds`, loop terminating early/late.
3. **Edge Case**: Failed on `[]`, `[1]`, negative numbers, etc.
4. **Syntax/Language**: Python specific errors (e.g. mutable default args, bad list slicing).
5. **Time limit**: Brute forced when an O(N) solution was needed.
