# Week 3: Stacks, Queues, and Linked Lists

## Overview
Welcome to Week 3! This week bridges the gap between primitive arrays and dynamic data structures. You'll move from contiguous memory (Lists and Strings) to structures built on nodes, references, and strict access patterns (LIFO/FIFO).

## Weekly Roadmap

- **Day 1: Stacks (LIFO) Basics:** Implementing a stack using Python lists, handling parentheses matching, and basic path resolution.
- **Day 2: Advanced Stacks:** Introduction to the Monotonic Stack pattern for "Next Greater Element" problems.
- **Day 3: Queues (FIFO):** Implementing Queues and Deques using `collections.deque` and applying them to BFS patterns.
- **Day 4: Linked Lists Fundamentals:** Building Singly and Doubly Linked Lists from scratch. Traversal, Insertion, and Deletion.
- **Day 5: Linked List Patterns:** Fast/Slow pointers (Cycle detection, middle node) and recursive/iterative Reversal.
- **Day 6: Marathon & Mock Test:** Stress-testing Week 3 concepts in a simulated environment.
- **Day 7: Mini-Project & Optimization:** Building a terminal-based Browser History simulator using Stacks and DLLs.

## Objectives
By the end of this week, you will be able to:
1. Identify when to enforce LIFO (Stack) vs FIFO (Queue) constraints.
2. Diagram and implement node reference updates without losing the head of a Linked List.
3. Optimize O(N^2) array problems down to O(N) using Monotonic Stacks.
4. Solve complex Linked List structural manipulations.
