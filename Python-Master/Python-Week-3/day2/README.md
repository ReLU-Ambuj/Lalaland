# Day 2: Advanced Stacks (Monotonic)

## Overview
A Monotonic Stack is a stack whose elements are always sorted (either entirely increasing or entirely decreasing). It is the premier pattern for solving "Next Greater Element" or "Daily Temperatures" problems in $O(N)$ time.

## Structure
- `notes/`: Deriving the Monotonic Stack pattern.
- `drills/`: Monotonic stack specific drills (`d01` to `d10`).
- `platform_problems/`: Medium-hard monotonic stack problems.
