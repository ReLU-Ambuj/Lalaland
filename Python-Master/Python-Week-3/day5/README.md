# Day 5: Adv. Linked List Patterns

## Overview
Mastering algorithms on Linked Lists involves clever pointer manipulation. The two most important patterns are: In-Place Reversal, and the Fast/Slow Pointer (Tortoise and Hare).

## Structure
- `notes/`: Reversal logic and cycle detection proofs.
- `drills/`: Reverse, Detect Cycle, Find Middle, Merge Sorted.
- `platform_problems/`: Medium hard pointers.
