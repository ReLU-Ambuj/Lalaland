# Day 4: Linked Lists Fundamentals

## Overview
Unlike arrays, Linked Lists do not store elements in contiguous memory. Each element (Node) stores its data and a pointer (reference) to the next Node. This allows $O(1)$ insertion/deletion if you have the pointer, but $O(N)$ search.

## Structure
- `notes/`: Memory models, building Node classes, edge cases (Head/Tail).
- `drills/`: Re-implementing iterations, adding, removing nodes.
